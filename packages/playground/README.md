# @santi020k/eslint-config-playground

Internal playground package for testing ESLint configurations against real files.

## Description

This package contains various file types (React, Astro, MDX, TS) to verify that the ESLint rules are applied correctly in a real-world project structure.

## Usage

Run linting on this package:
```bash
npm run lint -w @santi020k/eslint-config-playground
```
